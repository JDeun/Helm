# Helm v0.6.5 — Adoption Docs And Local Dashboard

**Date:** 2026-04-26

Helm 0.6.5 improves first-run adoption, OpenClaw/Hermes-style integration guidance, and local operational visibility.

## Added

- `helm status --brief` for a compact health-oriented status summary
- `helm dashboard` for a local text dashboard covering task state, checkpoints, memory review pressure, and next actions
- `helm report --format html` for a portable HTML operations report
- integration docs for OpenClaw, OpenHands-style long-lived runtimes, and existing agent workspaces
- comparison docs clarifying how Helm differs from agent frameworks, observability tools, and eval tools
- profile-template and demo scenario docs

## Changed

- README and README.ko are now shorter landing-style guides with sharper product copy, installer-first Quickstart, and links to deeper docs
- README and README.ko clarify that Helm is primarily for OpenClaw/Hermes-style long-lived agent workspaces, not one-off command-line tools
- release checklist now includes brief status, dashboard, and HTML report smoke checks
