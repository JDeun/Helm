# Helm v0.6.7 — Public Project Polish

**Date:** 2026-04-27

Helm 0.6.7 improves the public first-use path after the initial PyPI release.
It adds clearer install-first README copy, landing-page links, PyPI metadata
links, and a lightweight three-minute demo GIF for faster evaluation.

- animated three-minute demo GIF for README and demo documentation
- terminal capture SVG asset for the same demo flow
- public launch checklist covering GitHub, PyPI, demo assets, announcements,
  and landing-page readiness

  installer path

  PyPI publish workflow
- README and README.ko link the Helm landing page
- PyPI project metadata now includes Landing, Documentation, and Security URLs
- release/status copy now points at v0.6.7
