# Helm v0.6.4 — Guard Audit Hardening

**Date:** 2026-04-26

Helm 0.6.4 is a patch release for command guard classification and guard-only audit runs.

## Fixed

- `command_guard` now treats shell write redirection without separating whitespace, such as `echo x>file`, as a write operation
- read-only profiles therefore block these redirected shell commands instead of allowing them as read-only work
- `run_with_profile --guard-json` now records a final `guard_audit` task-ledger event before exiting without executing the command
