# Helm v0.6.3 — Conversational Capture Hotfix

**Date:** 2026-04-25

Helm 0.6.3 is a patch release for the v0.6.2 conversational capture flow.

## Fixed

- `helm memory capture-chat` now sets the task status before durable capture planning
- completed conversational tasks therefore produce the correct `memory_capture.finalization_status` of `capture_planned`
